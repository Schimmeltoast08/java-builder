# Java Builder
Java Builder, jbuild for short, is a tiny script to turn any .java file into a binary-like runnable file.

# Install
```
git clone https://github.com/Schimmeltoast08/java-builder.git
cd java-builder
./install.sh
```

# Usage
To use jbuild, run 
```
jbuild {filename}
```
**WITHOUT** the .java file extension.
For example helloworld.java --> jbuild helloworld
